#    Configuration management
-----------------------------

Today, we were introduced to configuration management, which is the process of automating the setup, maintenance, and updates of computer systems and software applications. It involves defining and enforcing consistent configurations across multiple machines, environments, and teams, so that they can work reliably and predictably.


### General Requirements
------------------------

- All your files will be interpreted on Ubuntu 20.04 LTS
- All your files should end with a new line
- A ```README.md``` file at the root of the folder of the project is mandatory
- Your Puppet manifests must pass ```puppet-lint``` version 2.1.1 without any errors
- Your Puppet manifests must run without error
- Your Puppet manifests first line must be a comment explaining what the Puppet manifest is about
- Your Puppet manifests files must end with the extension ```.pp```
